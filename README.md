## Laravel-InertiaJS-ReactJS

- [Laravel](https://laravel.com/docs/9.x/installation)
- [InertiaJS](https://inertiajs.com/)
- [VUE 3](https://vuejs.org/)

```bash
$ composer install
$ npm install
$ cp .env.example .env 
$ php artisan key:generate
# Configur your mysql connection
$ php artisan migrate
$ npm run dev
$ php artisan serve
```
http://127.0.0.1:8000/#
